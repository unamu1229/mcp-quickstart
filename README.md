# mcp-quickstart

## セットアップ

1. 依存パッケージのインストール

```
npm install
```

2. ビルド

```
npm run build
```

3. 実行

```
npm start
```

## 構成
- `src/` : ソースコード
- `index.ts` : エントリーポイント

## 必要条件
- Node.js (推奨バージョン: 18以上)
- npm
